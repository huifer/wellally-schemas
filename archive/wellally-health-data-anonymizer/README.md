# WellAll Health Data Anonymizer

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L5 privacy/compliance tool for de-identifying and anonymizing health data while preserving analytical utility.

## Capabilities
- PII detection and masking.
- Configurable anonymization (suppression, generalization, tokenization).
- Re-identification risk controls and reversibility policies.
- Batch processing for pipelines.

## Data Coverage
- Identity details
- Contact info
- Precise location
- Sensitive health fields

## Use Cases
- De-identifying clinical datasets before external sharing or model training.
- Creating safe analytics sandboxes with reversible tokenization policies.
- Compliance-by-design pipelines for cross-border data transfers.

## Compliance Targets
- GDPR
- HIPAA
- Applicable local data-protection regulations
