# WellAll 健康数据匿名化工具

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L5 隐私/合规工具，用于在保持可用性的前提下对健康数据进行脱敏与匿名化。

## 能力
- 识别并屏蔽 PII。
- 可配置的匿名化策略（删减、泛化、令牌化等）。
- 反识别风险控制与可逆性策略。
- 支持流水线的批量处理。

## 覆盖数据
- 身份信息
- 联系方式
- 精确位置
- 敏感健康字段

## 应用场景
- 模型训练或外部共享前的数据脱敏处理。
- 构建可逆/不可逆策略并存的安全分析沙箱。
- 跨境传输的数据合规预处理流水线。

## 合规目标
- GDPR
- HIPAA
- 适用的本地数据保护法规
