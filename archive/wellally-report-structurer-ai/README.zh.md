# WellAll 报告结构化 AI

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L3 AI 辅助抽取器，将非结构化医疗文本转成结构化字段，明确非诊断用途。

## 能力
- 关键字段识别与抽取。
- 带置信度的结构化 JSON 输出。
- 支持人工校验/回写流程。

## 场景
- 病历文本整理
- 报告字段提取
- 数据录入辅助与归档
- 需提交标准化字段的注册登记/报送自动化

## 声明
仅进行数据结构化，不提供任何诊断建议或结论。
