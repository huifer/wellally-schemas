# WellAll Report Structurer AI

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L3 AI-assisted extractor that turns unstructured medical text into structured fields. Non-diagnostic by design.

## Capabilities
- Key-field recognition and extraction.
- Structured JSON output with confidence scores.
- Supports review/override loops for human validation.

## Use Cases
- Clinical note tidying.
- Report field extraction.
- Data entry assistance and archival.
- Automating registry submissions where structured fields are required.

## Disclaimer
Provides data structuring only; no diagnostic advice or conclusions.
