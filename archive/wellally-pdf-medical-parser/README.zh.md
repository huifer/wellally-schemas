# WellAll 医疗 PDF 解析器

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L2 工具，用于解析医疗相关 PDF，并提取结构化字段供下游 Schema 使用。

## 功能
- PDF 文本提取与版面感知解析。
- 关键字段识别（抬头、章节、标识符）。
- 表格抽取。
- 输出结构化 JSON。

## 支持文档示例
- 门诊病历
- 住院记录
- 影像/检验报告
- 体检报告

## 应用场景
- 从历史 PDF 档案抽取结构化字段以便迁移。
- 解析医院门户可下载的报告，转换为标准化数据集。
- 作为实体抽取/摘要管线的前置步骤。
