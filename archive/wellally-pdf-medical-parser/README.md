# WellAll PDF Medical Parser

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L2 utility that parses medical PDFs and extracts structured fields for downstream schemas.

## Capabilities
- PDF text extraction and layout-aware parsing.
- Key field detection (headers, sections, identifiers).
- Table extraction.
- Structured JSON output.

## Supported Documents (Examples)
- Outpatient records
- Inpatient records
- Imaging/lab reports
- Health check reports

## Use Cases
- Extracting structured fields from legacy PDF archives for migration.
- Parsing downloadable hospital portals into normalized datasets.
- Pre-step before entity extraction / summarization workflows.
