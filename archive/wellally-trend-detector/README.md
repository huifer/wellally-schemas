# WellAll Trend Detector

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L3 AI-assisted tool that analyzes health metric trends and surfaces non-diagnostic insights.

## Capabilities
- Trend calculation and rate-of-change analysis.
- Anomaly/volatility detection.
- Forecasting (exploratory) and comparative views.
- Visual outputs (charts, heatmaps, reports).

## Analysis Types
- Long-term trends
- Periodicity
- Change-point detection
- Correlation hints (non-causal)

## Use Cases
- Wellness apps surfacing metric momentum (improving/worsening) to users without diagnosis.
- Provider dashboards tracking chronic disease indicators over months/years.
- Research teams monitoring cohort-level shifts after interventions.

## Disclaimer
Provides data trends only; not for diagnosis or treatment decisions.
