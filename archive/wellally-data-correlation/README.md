# WellAll Data Correlation

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L3 AI-assisted project exploring correlations between health metrics and behavior data. Non-causal, non-diagnostic.

## Capabilities
- Multi-dimensional correlation analysis.
- Pattern discovery and hypothesis surfacing.
- Time-series and cohort comparisons.
- Visual summaries for exploration.

## Data Types
- Physiological metrics
- Behavior/activity data
- Environmental factors
- Lifestyle signals

## Methods
- Statistical correlation
- Time-series analysis
- Clustering
- Machine learning feature importance

## Use Cases
- Exploring lifestyle factors correlated with changes in lab metrics.
- Product experimentation (A/B or cohorts) to see metric movement associations.
- Research discovery workflows to generate testable hypotheses.

## Disclaimer
Finds correlations only; does not establish causation or provide medical advice.
