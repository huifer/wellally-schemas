# WellAll 相关性分析

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L3 AI 辅助项目，用于探索健康指标与行为数据之间的相关性，明确非因果、非诊断用途。

## 能力
- 多维度相关性分析。
- 模式识别与假设提示。
- 时间序列与人群对比。
- 面向探索的可视化摘要。

## 数据类型
- 生理指标
- 行为/活动数据
- 环境因素
- 生活方式信号

## 方法
- 统计相关性
- 时间序列分析
- 聚类分析
- 机器学习特征重要性

## 应用场景
- 探索生活方式因素与检验指标变化的关联。
- 产品实验（A/B 或分组）观察指标波动的关联性。
- 研究发现流程中生成可验证的假设。

## 声明
仅发现相关性，不建立因果关系，不提供医疗建议。
