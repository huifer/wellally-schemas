# WellAll Lab Parser

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L2 utility that converts OCR’d lab report text into structured JSON.

## Capabilities
- Parse lab report text and extract metrics.
- Normalize values and units.
- Flag abnormal ranges.
- Output JSON aligned with WellAll lab schema.

## Use Cases
- Hospital OCR pipelines converting scanned lab slips into structured feeds.
- Health check centers digitizing bulk historical PDFs.
- Preprocessing step before ingestion into L1 lab-report schema for analytics.

## Input Formats
- Lab reports
- Health check reports
- General test result slips
- Raw OCR text
