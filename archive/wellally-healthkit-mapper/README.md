# WellAll HealthKit Mapper

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L2 utility that maps Apple Health (HealthKit) data into WellAll’s canonical schemas.

## Capabilities
- Read HealthKit samples and metadata.
- Map HealthKit types to WellAll schema fields.
- Normalize units and timestamps.
- Support incremental sync and backfills.

## Covered Data (Examples)
- Vitals and measurements
- Workouts and activity
- Sleep
- Nutrition (where available)

## Use Cases
- Consumer apps syncing HealthKit data into a unified health lake.
- Patient-controlled data exports for clinical onboarding.
- Research cohorts collecting BYOD (bring your own device) data feeds.
