# WellAll Consent Model

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L5 privacy/compliance model to manage the consent lifecycle for health data access.

## Capabilities
- Consent capture, versioning, and lifecycle tracking.
- Fine-grained access control and scopes.
- Revocation and emergency override workflows.
- Audit-ready consent history.

## Consent Types
- One-time consent
- Time-bound consent
- Revocable consent
- Granular, data-scope consent

## Compliance Support
- Consent record retention
- Right to know and portability
- Right to be forgotten

## Use Cases
- Patient-facing consent capture with granular scopes and expirations.
- Data-sharing hubs enforcing consent at query time.
- Emergency override workflows with full auditability.
