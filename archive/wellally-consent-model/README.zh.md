# WellAll 授权模型

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L5 隐私/合规模型，用于管理健康数据访问的授权生命周期。

## 能力
- 授权获取、版本管理与生命周期追踪。
- 细粒度访问控制与权限范围定义。
- 授权撤销与紧急覆盖流程。
- 符合审计要求的授权历史记录。

## 授权类型
- 一次性授权
- 期限性授权
- 可撤销授权
- 数据范围可配置的细粒度授权

## 合规支持
- 授权记录留存
- 知情权与数据可携权
- 被遗忘权

## 应用场景
- 面向患者的授权收集，支持细粒度范围与有效期。
- 数据共享/交换平台在查询时按授权实时过滤。
- 紧急覆盖流程的全程留痕与审计。
