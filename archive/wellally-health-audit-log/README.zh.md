# WellAll 医疗审计日志

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L5 隐私/合规项目，用于记录并管理医疗数据的访问与操作，提供防篡改审计能力。

## 能力
- 访问与操作日志（读取/写入/权限变更）。
- 实时监控与异常告警。
- 行为分析与风险评分。
- 符合监管要求的报告输出。

## 安全保障
- 日志完整性与防篡改机制。
- 加密存储。
- 对审计数据本身的访问控制。

## 应用场景
- EMR/PHI 系统的数据访问合规留痕。
- 安全运营监控高权限或异常的数据访问行为。
- B2B 健康平台向客户提供的可追溯审计记录。
