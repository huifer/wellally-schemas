# WellAll 单位归一化工具

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L2 工具，用于医疗检验单位的标准化转换（如 mg/dL ↔ mmol/L），确保数据一致性。

## 功能
- 单位识别与解析。
- 可追溯的双向转换。
- 面向流水线的批量转换。
- 转换验证，防止错误换算。

## 示例
- 血糖：mg/dL ↔ mmol/L
- 胆固醇：mg/dL ↔ mmol/L
- 甘油三酯：mg/dL ↔ mmol/L
- 其他生化指标按需扩展

## 应用场景
- 混合单位的检验数据在汇总/分析前先行标准化。
- 将设备读数与临床检验标准对齐。
- 机器学习流水线的预处理步骤，确保单位一致性。
