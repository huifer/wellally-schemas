# WellAll Anomaly Flagger

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L3 AI-assisted tool that detects anomalies and data-quality issues in health data pipelines.

## Capabilities
- Outlier detection and adaptive thresholds.
- Data-quality scoring and error tagging.
- Missing/format/logic checks.
- Auto-suggested fixes to support cleaning.

## Detection Focus
- Numeric outliers
- Logical inconsistencies
- Missing or sparse fields
- Format and unit errors

## Use Cases
- ETL/ELT quality gates before landing data into the health lake.
- Continuous monitoring of streaming vitals or device data with alerting hooks.
- Backfills to retro-audit legacy datasets prior to model training or analytics.
