# WellAll 异常标记器

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L3 AI 辅助工具，用于识别健康数据中的异常值与数据质量问题。

## 能力
- 异常值检测与自适应阈值。
- 数据质量评分与错误标记。
- 缺失/格式/逻辑一致性检查。
- 提供修复建议以协助数据清洗。

## 检测重点
- 数值异常
- 逻辑不一致
- 缺失或稀疏字段
- 格式与单位错误

## 应用场景
- 数据入湖/入仓前的质量闸口与自动拦截。
- 设备或连续监测数据的实时异常检测与告警。
- 历史数据回填的质量审计，支撑分析与建模。
