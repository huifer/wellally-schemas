# WellAll 辐射剂量计算器

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L4 计算模型，用于计算并跟踪 CT 检查的累积辐射剂量，明确非诊断用途。

## 功能
- 基于设备提供的 DLP 计算有效剂量。
- 按患者汇总累积剂量。
- 安全阈值预警与报告生成。
- 历史趋势分析。

## 计算依据
- DLP → 有效剂量转换。
- 不同部位权重因子。
- 年龄校正。
- 参考国际辐射剂量指南。

## 应用场景
- 影像中心跟踪患者跨检查的累积剂量。
- 质控与安全团队监测是否超阈值并预警。
- 面向患者的剂量“护照”，提升透明度。

## 声明
仅计算剂量数值，不评估医疗风险，也不替代临床决策。
