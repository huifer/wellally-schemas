# WellAll Medical Timeline

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

L2 utility to build an interactive timeline of personal medical events.

## Capabilities
- Chronological ordering and grouping of events.
- Multi-dimensional views (by category, provider, encounter).
- Correlated event grouping and drill-down details.
- Exportable timeline outputs.

## Event Types (Examples)
- Encounters/visits
- Labs and imaging
- Medications
- Symptoms and observations

## Use Cases
- Patient-facing longitudinal health view inside apps/portals.
- Clinician timeline to speed chart review and surface gaps.
- Research visualization for cohort event patterns.
