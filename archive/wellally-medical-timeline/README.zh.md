# WellAll 医疗时间线

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L2 工具，用于构建个人医疗事件的时间线，并支持可视化与分析。

## 功能
- 事件按时间排序与分组。
- 多维度视图（按类别/机构/就诊等）。
- 关联事件聚合与详情钻取。
- 时间线导出。

## 事件类型示例
- 就诊记录
- 检查与检验
- 用药记录
- 症状与观察

## 应用场景
- 面向患者的纵向健康档案视图（APP/门户）。
- 医护端时间线，提升病历回顾效率并提示缺口。
- 研究场景下展示队列事件模式的可视化。
