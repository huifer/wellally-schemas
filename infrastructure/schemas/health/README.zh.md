# WellAll 个人健康总 Schema

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

权威的 L1 JSON Schema，定义个人健康数据的顶层结构与基础数据类型，为全生态提供统一契约。

## 范围
- 面向个人健康数据的标准化资源结构。
- 供下游 Schema 复用的基础类型定义。
- 面向 API、存储与数据交换的互操作性保证。
- 需要时支持多语言字段标签/别名。

## 使用场景
- 各类健康应用的基础数据契约。
- API 负载格式与数据库表结构设计。
- 数据交换、集成与清洗管道。

## 状态
规划中。

## 贡献
