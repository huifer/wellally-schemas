# WellAll Health Master Schema

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

Authoritative Layer-1 JSON Schema that defines the top-level structure and core data types for personal health data across WellAll.

## Scope
- Canonical resource shapes for person-centric health data.
- Shared primitive definitions used by downstream schemas.
- Interoperability guarantees for APIs, storage, and exchange.
- Multi-language field labels/aliases where applicable.

## Use Cases
- Baseline contract for all health-related applications.
- API payload formats and database schema design.
- Data exchange and integration pipelines.

## Status
Planned.
