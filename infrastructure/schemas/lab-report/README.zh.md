# WellAll 生化检验报告 Schema

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L1 基础设施 Schema，标准化生化/检验指标、单位与参考区间。

## 范围
- 常见检验指标的标准化定义。
- 单位归一化（兼容 UCUM 表达）。
- 参考区间按年龄/性别等上下文区分。
- 异常/高/低/危急值标记机制。

## 覆盖
- 血常规
- 生化全项
- 电解质
- 肝肾功能
- 血脂与血糖
- 甲状腺功能

## 状态
规划中。
