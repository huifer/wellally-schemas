# WellAll Lab Report Schema

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

Layer-1 schema for biochemistry and lab reports, standardizing metrics, units, and reference ranges.

## Scope
- Canonical definitions for common lab metrics.
- Unit normalization and UCUM-friendly representations.
- Reference ranges contextualized by age/sex.
- Flags for abnormal/high/low/critical values.

## Coverage
- CBC
- Comprehensive metabolic panel
- Electrolytes
- Liver/kidney function
- Lipids and glucose
- Thyroid function
