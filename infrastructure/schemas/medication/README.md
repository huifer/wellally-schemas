# WellAll Medication Schema

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

Layer-1 schema for standardized medication records, ensuring consistent drug naming, dosing, and scheduling.

## Scope
- Ingredient/brand naming normalization.
- Dosage form, strength, route, frequency, and timing.
- Prescription context (prescriber, encounter, refills).
- Interaction flags and classification codes (e.g., RxNorm/ATC).
