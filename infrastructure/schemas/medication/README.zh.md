# WellAll 药物记录 Schema

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L1 基础设施 Schema，规范药物记录的数据结构，确保药名、剂量与给药计划的一致性。

## 范围
- 通用名/商品名的标准化表示。
- 剂型、剂量、给药途径、频次与时间。
- 处方上下文（开方医生、就诊信息、续方次数）。
- 相互作用标记与分类编码（如 RxNorm/ATC）。

## 状态
规划中。
