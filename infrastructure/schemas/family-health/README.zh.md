# WellAll 家庭健康 Schema

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L1 基础设施 Schema，用于描述家庭关系与家族健康史（“家庭健康树”）。

## 范围
- 代际、直系与旁系关系建模。
- 家族病史与遗传风险标记的记录。
- 家庭成员档案及关联记录。
- 可选的关系图衍生洞察（非诊断）。

## 关系类型
- 直系亲属
- 旁系亲属
- 代际关系
- 遗传标记（如适用）
