# WellAll 影像报告 Schema

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

L1 基础设施 Schema，用于规范 CT/超声/MRI/X 光/PET-CT 等影像报告的数据结构。

## 范围
- 报告元信息（机构、时间、设备、部位）。
- 扫描技术与协议描述。
- 影像所见、测量值与结构化观察字段。
- 诊断印象/总结字段。

## 支持模态
- CT
- 超声（B 超）
- MRI
- X 光
- PET-CT

## 状态
规划中。
