# WellAll 健康 JSON 规范

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)

本白皮书是 WellAll 生态的 L1 核心规范，定义健康数据的 JSON 格式与验证规则。

## 核心内容
- JSON 结构与格式规范。
- 数据类型定义与字段命名约定。
- 校验规则与兼容性原则。
- 编写最佳实践与扩展指引。

## 规范范围
- 标准字段命名与分层。
- 必填/可选字段及校验语义。
- 扩展位与厂商特定字段的兼容策略。
