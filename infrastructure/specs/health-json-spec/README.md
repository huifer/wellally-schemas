# WellAll Health JSON Spec

[![中文](https://img.shields.io/badge/Language-中文-red)](README.zh.md)

Canonical Layer-1 whitepaper that defines the JSON format for health data across the WellAll ecosystem.

## What’s Inside
- JSON structural and formatting rules.
- Data type definitions and naming conventions.
- Validation requirements and compatibility principles.
- Authoring best practices and extensibility guidance.

## Scope
- Standard field naming.
- Required/optional fields and validation semantics.
- Extension hooks for vendor-specific data while keeping interoperability.
